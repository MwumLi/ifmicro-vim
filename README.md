# ifmicro-vim

## Quick install

    $ git clone -b light https://github.com/MwumLi/ifmicro-vim.git
    $ cd ifmicro_vim && make install 

## Quick update  

    $ make update
    
## 目标

轻快  
如果已经影响 vim 的速度, 那么这样的配置宁愿不用  

1. 一个配色看起来不错, 用起来顺手的编辑器
